## ✨ Outstanding Features

- **📂 Multi-Collection System:** Create isolated knowledge bases dynamically. Your "Legal" AI chatbot won't hallucinate facts mixed up from your "Medical" documents!
- **⚡ Bulk Drag-and-Drop Uploads:** Easily upload multiple PDF files simultaneously. The pipeline elegantly parses, chunks, and batch-indexes them in the background with a unified progress bar.
- **🧠 Local Vector Store (FAISS):** Blazingly fast, privacy-focused, and CPU-friendly similarity search without relying on expensive monthly hosted vector databases.
- **🔍 Source-Referenced Answers:** Trust but verify. Every contextual response generated by Gemini provides an expandable UI section revealing the exact source snippets it used from your PDFs.
- **🗃️ Smart Metadata Tracking:** Integrated with SQLite to automatically track document chunk ranges and prevent duplicate document processing via SHA-256 hashing.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/rag-chatbot.git
cd rag-chatbot
```

### 2. Set up the Environment
Create a clean virtual environment and activate it:
```bash
python -m venv venv

# On Windows Cmd / Powershell:
venv\Scripts\activate

# On macOS / Linux:
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure API Keys
Create a `.env` file in the root directory and securely add your Google Gemini API key:
```env
GEMINI_API_KEY=your_actual_api_key_here
```

### 5. Launch the Application
```bash
streamlit run app.py
```

## 📚 How to Use

1. **Launch the web application** to open the interface in your browser.
2. In the Sidebar, locate the **➕ Create New Collection** box and name your topic (e.g., "Finance Regulations").
3. Use the elegant **Drag-and-Drop file uploader** to deposit one or *multiple* PDF files at once.
4. Hit **Process Document(s)**. Sit back as the application calculates the chunks and indexes them incrementally.
5. Once processing reaches 100%, step into the main chat window and **fire away your questions!**
